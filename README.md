## Create a virtual environment :

(Windows)
```
python -m venv env
```

(MacOS)
```
python3.11 -m venv env
```

## Activate the virtual environment :

```
source env/bin/activate
```

## Installation:
(Windows)
```
pip install -r requirements.txt
pip install langchainhub
```

(MacOS)

```
pip3 install -r requirements.txt
```
